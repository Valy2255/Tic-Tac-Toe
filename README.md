## **How to Run the Project**

1. Clone the repository:
   ```bash
   git clone https://github.com/Valy2255/Tic-Tac-Toe.git
   cd Tic-Tac-Toe
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start dev
   ```
4. Open the app in your browser.
